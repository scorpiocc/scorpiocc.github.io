亚马逊VPS服务器搭建

发表于 2017-08-08 | 分类于 亚马逊服务器 | | 阅读次数
字数： 1,087

作为一个程序员，肯定要经常使用谷歌查询资料，但是网络上大部分的免费帐号又是延迟很严重，买独享帐号又要花费很多。正好亚马逊有免费1年的服务器使用(每月15G流量,超出会扣费)，此教程仅为个人理解，欢迎提出意见。

第一步：注册

传送门：注册
别选错了，选我是新用户，输入邮箱按登陆，会进入下一个页面进行注册。
——-注册过程很简单，自己填写，输入用户名，密码——-
进入下一步后选择个人，别选择公司，再下一步需要绑定储蓄卡或信用卡，不是所有的储蓄卡都支持，本人绑定的浦发信用卡，手续费要扣除2美金（可用RMB还款，不过14块用1年独享帐号很舒服了是吧=。=）

第二步:登陆亚马逊控制台，创建VPS

注册完事了登陆到控制台：选择EC2，没有就直接搜索EC2选择第一个（云中的虚拟服务器）

如图：选择启动实例 右上角（选择日本东京，相对而言快一点）

如图：选择Ubuntu Server 16.04
(如果碰到读取AIM失败，刷新下，开着代理的话把代理关掉！)

选择完之后直接按照默认设置下一步，直到配置安全组，点击添加规则（安全起见，懂的人可以自行开放ss需要的端口，不懂的就按照如图所示配置）

最后：按右下角的审核和启动 再最后按 启动。
重点来了！！！

第三步：登陆VPS，安装ss后台。

下载 Xshell 5 ：地址:6hhr
如果失效或无法安装，请自行百度官网下载
打开Xshell 新建会话 名字随意 主机填写 刚才新建的实例 的公有IP（控制台的IP）

用户身份验证
方法选择 Pulic key 用户名 ubuntu 用户密钥：浏览 导入刚才保存的 密钥对

如果跳出如下界面，则连接成功。（可能会等待一会儿，请自行调试）

登陆以后执行以下命令
sudo passwd root 此命令输入完之后会提示让你输入 root的密码，设置一个记住就好，会让你输入两遍（如果忘记密码，可重复执行）

sudo chmod 777 /etc/ssh/sshd_config 更改文件权限为777
vi /etc/ssh/sshd_config 执行完此命令后 在英文状态下按 字母i 进入编辑模式(进入编辑模式后键盘↓箭头按住可向下查看代码。)

改完之后，按ESC 输入:wq 保存并退出 :wq是完整的命令 前面有冒号
输入 su root 输入刚才设置的密码 进入root用户，输入命令 reboot 重启vps

重启需要1-2分钟。 之后重新打开软件，修改用户配置，点击属性

帐号修改为之前设置的root帐号

点击连接，如果出现如图所示，则登陆成功，不然可能是之前编辑模式没有修改好，或者其他操作失误。

第四步，安装ss服务（重点！）

最后一条密令执行完事，会提示输入密码（之后ss帐号密码） 然后提示输入端口 可默认，最后任意键继续。如果碰到以下情况删除不了的，按如下操作,有基础的请无视
然后一直回车就可以了，接着代码滚屏。

最终修成正果 Enjoy it !

PS：此服务器一个月只免费750小时，1个实例一个月(720小时足够)，2个就只能用半个月了，请注意

打开Shadowsocks输入帐号密码端口，确定，即可。
